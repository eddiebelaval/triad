# /debt - Derived Debt Report from the Triad

You are computing a **debt report** (technical + product) from the three-document triangle (VISION.md, SPEC.md, BUILDING.md).

Load and follow the skill at: `.claude/skills/triad-derive/SKILL.md` — execute the `/debt` command section.

## Arguments
$ARGUMENTS

## Quick Reference
- No arguments: full debt analysis for current project
- `--project <name>`: target a specific project
- `--type tech`: only technical debt
- `--type product`: only product debt
