# /pitch - Derived Pitch Narrative from the Triad

You are computing a **pitch narrative** from the three-document triangle (VISION.md, SPEC.md, BUILDING.md).

Load and follow the skill at: `.claude/skills/triad-derive/SKILL.md` — execute the `/pitch` command section.

## Arguments
$ARGUMENTS

## Quick Reference
- No arguments: full pitch (short + long) for current project
- `--project <name>`: target a specific project
- `--format investor`: investor-focused framing
- `--format user`: user-facing / landing page framing
- `--format grant`: grant application framing
